# react_game_hub

A Game Website built using React.js. It displays games fetched using an API from [rawg.io](https://rawg.io/). Visitors to the website can filter the games based on genre and platform and sort based on different criteria as well. They can also use the Search bar to search for a game with a aparticular keyword. The project does not contain any backend.

To view live website, please click [here](https://react-game-hub-ecru.vercel.app/).
